## Описание

Реализация HTML+CSS проект гистограммы, отображающей доходность предприятия по месяцам с использованием Flexbox.
Задача взята из учебного практического курса.

## Требования к задаче 
Технические ограничения:
Запрещено использовать display: grid и связанные с Grid свойства. Задание направлено на изучение исключительно Flexbox-позиционирования.
Запрещено использовать фиксированные, абсолютные и относительные позиционирования

Основные требования:
В проекте должно быть использование display: flex как минимум 3 раза
Адаптивность не предусматривается. Тестирование будет происходить на экране с шириной 1100px
